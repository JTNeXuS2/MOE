### Установка
Скачать RunAsDate.exe и BattleField_Looper.bat
Поместить в директорию C:\вашкластер\MatrixServerTool
отредактировать BattleField_Looper.bat подставив параметры запуска даты (https://discord.com/channels/786062570095116293/1171345133141237780/1216977618021388328)

также незабудьте указать ид вашего сервера (7020) в строке 
set "serverid=7020"

### Installation
Download RunAsDate.exe and BattleField_Looper.bat
Place in the directory C:\yourcluster\MatrixServerTool
edit BattleField_Looper.bat by substituting launch parameters, set date (https://discord.com/channels/786062570095116293/1171345133141237780/1216977618021388328)

also don’t forget to indicate your server ID (7020) in the line
set "serverid=7020"
